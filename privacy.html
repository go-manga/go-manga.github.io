GoManga 隱私政策

最後更新：2026 年 6 月 20 日

摘要

GoManga 是本機優先的 iOS 漫畫閱讀器。GoManga Team 不經營 GoManga 帳號系統、不代管漫畫內容，也不會在本網站加入第三方 analytics 或自動翻譯腳本。

你設定的來源、書庫、閱讀進度、清單、偏好與診斷資料主要保存在你的裝置上。只有當你主動提交 issue、截圖、診斷片段或其他支援資訊時，這些內容才會離開你的裝置。

保存在裝置上的資料

GoManga 可能在裝置上保存以下資料，用於提供閱讀與續讀體驗：

  • 書庫、稍後閱讀、閱讀歷史與閱讀進度。
  • 你加入或安裝的來源紀錄，以及來源相關設定。
  • app 偏好設定，例如語言、外觀、閱讀方向與快取設定。
  • 封面、章節頁面與 metadata 快取，用於提升載入速度與重讀體驗。
  • 本機診斷記錄，用於追查 app 行為、錯誤與效能問題。

來源與網路連線

GoManga 會依照你自己的設定連線到來源、來源服務或本機服務，以取得清單、詳情、章節與圖片頁。這些第三方網站、服務或你自己的本機服務不由 GoManga Team 控制，並可能依其自身政策處理你的連線資訊。

iOS 可能會詢問本機網路權限，原因是 GoManga 需要連線到你設定的本機來源服務。你可以在 iOS 系統設定中管理這項權限。

網站與 GitHub

本網站以 GitHub Pages 提供，問題回報使用 GitHub Issues。GoManga Team 不在本網站加入 Google Analytics、廣告 SDK 或第三方行為追蹤。GitHub 仍可能依 GitHub 自身政策處理標準伺服器記錄、登入狀態與 issue 內容。

如果你使用 GitHub issue 或其他第三方平台提交回饋，該平台可能會記錄帳號、IP、瀏覽器資訊與提交內容。公開 issue 會被其他人看見。

診斷與問題回報

當你透過 GoManga 問題回報、GitHub issue、電子郵件或其他方式主動提交問題時，你提供的內容可能包含 app 版本、iOS 版本、裝置型號、重現步驟、來源 host、錯誤訊息、截圖或診斷片段。

請不要提交帳號密碼、私密 token、完整 cookie、付費內容、個人通訊資料或其他敏感資訊。若需要分享診斷資料，請先檢查並移除不該公開的內容。

兒童隱私

GoManga 不是專為兒童設計的服務，也不會刻意收集兒童個人資料。若你認為問題回報中誤含未成年人的個人資料，請聯絡我們移除。

政策變更

我們可能因 app 功能、支援流程或法規需求更新本政策。更新後會調整本頁的最後更新日期。

聯絡方式

若你有隱私相關問題，請透過 問題回報頁 建立 issue，或使用 GitHub repository 提供的聯絡方式。
