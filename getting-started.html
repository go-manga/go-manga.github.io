Getting Started

GoManga 上手指南

這份指南只涵蓋 GoManga app 自己負責的流程；來源內容、帳號、授權與可用性由你設定的來源或服務決定。

1. 先加入 Repositories

GoManga 的來源流程從 Repositories 開始。加入 repository 後，app 才能顯示可安裝或可使用的 Sources。

GoManga 不提供官方內容目錄，也不推薦特定第三方來源。請只加入你信任且有權使用的 repository 或本機服務。

2. 安裝或啟用 Sources

Sources 是你用來瀏覽目錄、搜尋、開啟詳情與載入章節的來源入口。若來源需要登入、mirror、token 或驗證，請在 app 內的 Source Settings 依來源要求設定。

3. 瀏覽、加入書庫或稍後閱讀

從 Browse 或來源 catalog 開啟作品詳情後，你可以加入書庫或稍後閱讀。書庫適合長期追蹤；稍後閱讀適合先保存但還不想放進正式書庫的作品。

4. 開始閱讀與續讀

從作品詳情、書庫、稍後閱讀或閱讀歷史都可以回到閱讀流程。GoManga 會保存閱讀進度，讓你下次從合適的位置繼續。

5. 遇到問題時

若來源需要驗證、章節空白、圖片載入失敗、進度沒有保存或 app 行為異常，請先記下發生位置、app 版本、iOS 版本、來源 host 與重現步驟，再到 回報問題 建立 issue。
